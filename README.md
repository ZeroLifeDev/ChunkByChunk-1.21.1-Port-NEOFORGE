# Chunk By Chunk - NeoForge 1.21.1 Port

This is a port of the [Chunk by Chunk](https://github.com/immortius/chunkbychunk) mod to Minecraft 1.21.1 using NeoForge.

## Features (Ported)
- **Sky Chunk Generation**: Dynamic world generation in a custom dimension.
- **Chunk Spawners**: Items to generate chunks in the sky.
- **World Scanner, Mender, and Forge**: Functional blocks for chunk management.
- **1.21.1 Native Support**: Updated to use modern registry handling and capabilities.

## Port Notes
- Replaced outdated Mixins with robust reflection-based registry unfreezing.
- Migrated legacy fluid handling to NeoForge's native capability system.
- Standardized data pack formats for 1.21.1 compatibility.
